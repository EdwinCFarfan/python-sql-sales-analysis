# 📊 Python SQL Sales Analysis

Proyecto de análisis de ventas utilizando Python y SQLite.

## 🎯 Objetivo

Conectarse a una base de datos SQLite desde Python para ejecutar consultas SQL y obtener métricas comerciales relevantes.

## 🗄️ Base de Datos

La base de datos contiene las siguientes tablas:

- clientes
- productos
- pedidos
- detalle_pedidos

## 📊 Análisis Realizados

### Clientes con mayor gasto

Se calcula el gasto total de cada cliente mediante consultas SQL con JOIN y SUM.

### Productos más vendidos

Se identifican los productos con mayor cantidad de unidades vendidas.

### Cantidad de pedidos por cliente

Se analiza la frecuencia de compra de cada cliente.

### Precio promedio de productos

Se calcula el precio promedio de los productos disponibles.
